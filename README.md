Greetings. Welcome to Nebunel Project.

**First steps to do as contributor**

1. Fork this project first and then add the linux project into this repo working folder (no clone or fork of linux, but copy of the whole source code).
2. Start working on the file system code.
3. Commit to your own branch where branch name is your own name.
4. Submit a pull request along with a notification about your commit in **Issues** Section.
